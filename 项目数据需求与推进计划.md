## 📊 **数据需求清单**

### **🔥 核心分析功能必需数据**

#### **1. 设施详细信息补充**
```javascript
// 数据用途: MapView.vue 3D地图的设施布局分析
// 获取难度: ⭐⭐ (简单)

需要补充的字段:
{
  "facility_id": "唯一标识",
  "name": "设施名称", 
  "type": "设施类型", // 已有
  "lng": "经度", // 已有
  "lat": "纬度", // 已有
  
  // 需要补充的字段

  "area": "建筑面积", // 大概的面积信息
  "address": "详细地址", // 已有但可能需要补充
  "phone": "联系电话",
  "open_time": "营业时间",
  "established_year": "建成年份" // 可选
}

数据量: 约1000个设施
获取方式: 
- 高德POI API查询
- 政府部门官网查询
- 电话核实 (可选)
```
